# JANITOR
## Joint Autonomous Navigation & Intelligent Terrain Operations Robots
Texas A&M Senior Capstone project sponsored by Los Alamos National Laboratories
Simulates Reeds-Shepp, Dubins, Differential Drive, and Center-Articulated vehicles in the cleanup process of Hanford nuclear waste storage tanks using C# in Godot.
