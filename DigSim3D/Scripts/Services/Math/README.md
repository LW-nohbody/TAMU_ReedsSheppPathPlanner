# Path Generation Math

Contains the files for generating paths both in high level view and as a series of points

+ [DDSampler](DDSampler.cs) - Converts high level Differnential Drive path into a series of points
+ [DSampler](DSampler.cs) - Converts high level Dubins path into a series of points
+ [RSSampler](RSSampler.cs) - Converts high level Reeds-Shepp path into a series of points
+ [DifferentialDrivePaths](DifferentialDrivePaths.cs) - Generates the high-level Differential Drive paths
+ [DubinsPaths](DubinsPath.cs) - Generates the high-level Dubins paths
+ [ReedsSheppPaths](ReedsSheppPaths.cs) - Generates the high-level Reeds-Shepp paths
