# UI Features

Defines all UI features

+ [AnimatedValueLabel](AnimatedValueLabel.cs) - Handles the value labels for the robot information ie)speed, dig depth, etc
+ [BufferVisualizer](BufferVisualizer.cs) - Displays the wall and obstacle buffers
+ [DigSimUI](DigSimUI.cs) - Main UI display for simulation information, creatse the UI panels
+ [MiniChart](MiniChart.cs) - Defines line-chart displayed in left panel of UI to specify each vehicle's filled-capacity over time
+ [PremiumRobotStatusEntry](PremiumRobotStatusEntry.cs) - Monitoring panel for robot status
+ [PremiumSlider](PremiumSlider.cs) - Slider used in UI for robot speed and dig configuration
+ [PremiumUIPanel](PremiumUIPanel.cs) - Panel definition for UI
+ [PresetButtonGroup](PresetButtonGroup.cs) - Creates buttons to set sliders and associated configurations to preset values
+ [TerrainHeightMapThumbnail](TerrainHeightMapThumbnail.cs) - Displays terrain heat map based on terrain height
+ [UIToggleSwitch](UIToggleSwitch.cs) - Creates switch to toggle UI visibility
