# Scripts

All scripts used in the system

+ [App](App/README.md) - Overall simulation controller
+ [SimulationConfig](Config/SimulationConfig.cs) - Configures the simulation details, ie) robot speed, dig depth, etc
+ [DebugPath](Debug/DebugPath.cs) - Prints debugging information for path generation
+ [Domain](Domain/README.md) - Defines various necessary aspects for simulation, ie) obstacles, vehicles, world, dig sites (includes priority assignment)
+ [Services](Services/README.md) - Service providers, calculates paths, decides tasks, and creates dig targets
+ [UI](UI/README.md) - UI features
